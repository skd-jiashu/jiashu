# 《家书》剧组官网项目大纲

## 网站结构概览

### 页面架构
1. **index.html** - 主页
2. **cast.html** - 演员阵容
3. **schedule.html** - 演出信息
4. **about.html** - 关于我们

### 共享组件
- **导航栏** - 统一的顶部导航
- **页脚** - 联系信息和版权
- **返回顶部** - 浮动按钮

## 页面详细规划

### 1. 主页 (index.html)
**目标**: 营造强烈的情感冲击，展现剧目的艺术价值

**内容结构**:
- **Hero区域**
  - 动态粒子背景效果 (p5.js)
  - 剧目标题和副标题
  - 核心宣传语
  - 主演剧照轮播 (Splide.js)

- **信件阅读器交互区域**
  - 多个信封图标展示
  - 点击展开家书内容
  - 手写体文字效果
  - 情感化背景切换

- **剧目亮点**
  - 三个核心特色的可视化展示
  - 使用ECharts.js创建数据可视化
  - 动画数字统计

- **观众评价**
  - 滑动卡片展示
  - 真实观众反馈
  - 星级评分可视化

**技术特效**:
- 滚动视差效果
- 文字逐字显现动画
- 图片Ken Burns效果
- 粒子飘散背景

### 2. 演员阵容 (cast.html)
**目标**: 深度展示演员与角色的情感连接

**内容结构**:
- **页面标题区域**
  - 简洁的标题和简介
  - 背景使用微妙的纹理效果

- **主要角色展示**
  - 左侧：演员照片网格 (3x3布局)
  - 右侧：角色详情面板
  - 点击演员照片切换角色信息
  - 包含角色介绍、演员感言、角色关系图

- **演员访谈**
  - 视频或音频访谈片段
  - 文字访谈内容
  - 演员与角色的共鸣故事

- **幕后花絮**
  - 排练照片轮播
  - 创作过程分享
  - 团队精神展示

**交互功能**:
- 演员照片悬停效果
- 角色信息动态切换
- 图片放大查看功能
- 社交分享功能

### 3. 演出信息 (schedule.html)
**目标**: 提供完整的演出安排和购票信息

**内容结构**:
- **演出时间轴**
  - 水平时间轴展示
  - 使用Matter.js创建物理交互
  - 点击日期显示详细信息

- **演出日历**
  - 月视图日历
  - 特殊场次标记
  - 快速导航功能

- **场次详情**
  - 演出时间、地点
  - 票价信息
  - 剩余票量可视化
  - 购票链接

- **演出历史**
  - 过往演出回顾
  - 观众反馈统计
  - 媒体报道链接

**数据可视化**:
- 演出场次统计图表
- 观众满意度雷达图
- 地域分布热力图

### 4. 关于我们 (about.html)
**目标**: 展现剧组的文化使命和艺术追求

**内容结构**:
- **剧组介绍**
  - 创作初衷和使命
  - 艺术理念阐述
  - 团队价值观

- **创作历程**
  - 时间线展示创作过程
  - 重要节点和突破
  - 团队成长故事

- **情感共鸣测试**
  - 交互式问答系统
  - 个性化结果展示
  - 社交分享功能

- **联系信息**
  - 剧组合照
  - 联系方式
  - 社交媒体链接

**特色功能**:
- 创作历程动画时间轴
- 团队成员介绍卡片
- 情感测试交互组件

## 技术实现计划

### 核心库集成
- **Anime.js**: 所有页面的动画效果
- **p5.js**: 主页粒子背景和创意效果
- **ECharts.js**: 数据可视化图表
- **Splide.js**: 图片轮播和滑动效果
- **Matter.js**: 演出时间轴的物理交互
- **PIXI.js**: 高性能视觉特效
- **Shader-park**: 背景着色器效果

### 响应式设计
- 移动端优先的设计理念
- 断点设置：320px, 768px, 1024px, 1440px
- 触摸友好的交互设计
- 图片自适应和懒加载

### 性能优化
- CSS和JS文件压缩
- 图片格式优化 (WebP支持)
- 关键资源预加载
- 非关键资源懒加载
